Os 3 Pilares

O HTML (HyperText Markup Language) define a estrutura de uma página. Ele organiza o conteúdo em títulos, textos, imagens, links, etc. É como o esqueleto de um site.

O CSS (Cascading Style Sheets) cuida do visual da página. Ele altera cores, fontes, tamanhos, posições e deixa o site bonito e responsivo. É como a decoração da casa.

O JavaScript adiciona interatividade à página. Ele responde a cliques, valida formulários, cria animações e atualiza conteúdos. É como a eletricidade da casa.

Desenvolvimento Web

O desenvolvimento web é o processo de criação de sites e aplicações para a internet. Envolve a construção da estrutura (HTML), o estilo visual (CSS) e a interatividade (JavaScript) de uma página. Esse campo pode incluir desde páginas simples até sistemas complexos, como redes sociais, lojas virtuais e plataformas web.

Profissões

As 3 Profissões mais famosas do desenvolvimento web

Front-End

Responsável pelo que o usuário vê e interage. Constrói a interface do site usando HTML, CSS e JavaScript.

Back-End

Cuida do que está por trás das cortinas. Trabalha com servidores, bancos de dados e lógica para fazer tudo funcionar.

UX/UI Design

Foca na experiência e aparência. Garante que o site seja fácil de usar, bonito e intuitivo para o usuário.
